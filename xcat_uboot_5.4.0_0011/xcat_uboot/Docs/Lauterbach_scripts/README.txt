This folder contains Lauterbach scripts for xCat boards:

xCat/xCat_dx2122-A1-NAND-320MHz.cmm 	- for xCat DB FE board, NAND support, 320MHz CPU freq.
xCat/xCat_dx2122-A1-SPI-320MHz.cmm	- for xCat DB FE board, SPI support, 320MHz CPU freq.
xCat/xCat_dx3121-A1-200MHz.cmm		- for xCat RD board, SPI support, 200MHz CPU freq.
xCat/xCat_dx4122-A1-NAND-320MHz.cmm	- for xCat DB GE board, NAND support, 320MHz CPU freq.
xCat/xCat_dx4122-A1-SPI-320MHz.cmm	- for xCat DB GE board, SPI support, 320MHz CPU freq.
This scripts enables full Lauterbach functionality, hw breakpoints should be used.

xCat2 boards:

xCat2/xCat2-dx2122-NAND-320MHz.cmm	- for xCat2 DB FE board, NAND support, 320MHz CPU freq.
xCat2/xCat2-dx2122-SPI-320MHz.cmm	- for xCat2 DB FE board, SPI support, 320MHz CPU freq.
xCat2/xCat2-dx4122-NAND-320MHz.cmm	- for xCat2 DB GE board, NAND support, 320MHz CPU freq.
xCat2/xCat2-dx4122-SPI-320MHz.cmm	- for xCat2 DB GE board, SPI support, 320MHz CPU freq.
xCat2/xCat2-A1-Qfp-SPI-200MHz.cmm	- for xCat2 RD GE board, SPI support, 200MHz CPU freq.
xCat2/xCat2-A1-Qfp-SPI-250MHz.cmm	- for xCat2 DB GE board, SPI support, 250MHz CPU freq.
xCat2/xCat2A0-trivial-no-DRAM-params.cmm- For all xCatA0 boards.
Note: For QFP boards use xCat2/xCat2A0-trivial-no-DRAM-params.cmmfor full debug with U-Boot.
